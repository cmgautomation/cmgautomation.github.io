Selection Program  for technical office

This window is for selecting a component and inserting it’s image in an AutoCAD drawing. This possibility speeds up the layout drafting and the executive designing of the plant.

The inserted sketches are Autocad blocks that can contain additional information for an automatic generation of a part list.

 

These are four icons for selecting 4 different sketches associated with the component (for example, the various views).

This is the icon for inserting the visualized sketch into an AutoCAD drawing..
