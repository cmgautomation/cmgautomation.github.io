Finishing Patent

A fast color change system without paint waste.

Italian patent N° M02002A000241

This patent offers a solution to the problem of quick color substitutions in industrial finishing plants. It is also a simple solution for the application of products with two or more components which have a very low mixture ratio.  This system makes it easy to manage electrostatic paints.

 

The plant is made up of a series of  modules (2...5).  Each one is dedicated to a different color or to a different kind of paint.  A conveyer (8) connects all of the paint modules with an application device (for example, a robot). The conveyer moves the cups (6) where the modules can dose the paint.  The first module (7) contains solvent and  automatically washes and cleans the cups in movement on the conveyer system.  If necessary, the last module of the line can dose the solvent and it can mix and automatically correct the paint viscosity.

 

There is an automatic paint gun on the robot with a fitting which permits the robot to automatically change an empty cup with a new one, connecting the new cup directly to the gun itself.  If necessary, the internal parts of the gun can be cleaned, after each cycle, using the solvent supplied by a hose that passes through the robot's arms.  In this way, when a new cup is inserted, the gun is clean and ready to be used with the new color.

 

If a plant uses many colors obtained from the combination of base colors, the computer that pilots the plant can produce and manage an unlimited amount of colors.  It can combine the base colors contained in the modules directly in the cups.  If a plant uses only a few colors, it is possible to have each of these colors, ready to use, in a different module (72).

 

On the modules there are volumetric instruments which control the dosing of the paint in the cups.  Therefore, the paint in each cup can be the exact amount needed for the application request.  In this way, it is possible to frequently change colors without any relevant paint and solvent waste.

 

When the paint has a catalytic component, one of the modules contains and doses the catalyst directly into the cups.  Inside each cup there is a propeller for the optimal mixing of the resins, solvent and catalyzer.  In this way, it is possible to treat products made up of two or more components, which have very low mixture ratios, without any problems.

When you must use an electrostatic paint, all you need to do is isolate the gun from the robot and, only when it is necessary, isolate the piping that brings solvent to the gun.
